# 🔍 Stock Finder - Real-Time Search App

## 📌 Project Overview

Stock Finder is an interactive web application that allows users to search and filter stock names in real time. As the user types in the search bar, the list dynamically updates to show matching results.

This project was built to demonstrate JavaScript logic, DOM manipulation, and user interaction handling.

---

## ✨ Features

* 🔎 Real-time search filtering
* ⚡ Instant UI updates while typing
* ❌ "No results found" handling
* 📱 Responsive design
* 🎯 Clean and minimal UI
* 🧠 Efficient JavaScript logic using event listeners

---

## ⚙️ How It Works

* User types into the search bar
* JavaScript captures input using `keyup` event
* Filters list items using `.includes()`
* Displays matching results dynamically
* Shows "No results found" if no match exists
